About Hyperspeed

Hyperspeed is a web proxy service that allows you to browse the internet securely and privately. It acts as an intermediary between you and the websites you visit, helping to bypass restrictions and maintain your privacy online.

Proxy Backends

Hyperspeed supports multiple proxy backends to route your traffic:

Ultraviolet Working

A highly sophisticated proxy used for evading internet censorship. It's currently the recommended backend for optimal performance and compatibility.

Scramjet Currently Broken

An alternative proxy backend that is currently experiencing technical issues. We're working to resolve these problems. Please use Ultraviolet for now.

WISP Server

WISP (WebSocket Internet Service Protocol) is used to establish secure connections through the proxy. The default server is wss://wisp.mercurywork.shop/ and should work for most users.

You can customize this if you have your own WISP server or need to use a different endpoint.

How to Use

  • Enter a search term or direct URL in the search bar
  • Select your preferred search engine
  • Choose a proxy backend (Ultraviolet recommended)
  • Verify the WISP server is set correctly
  • Click "Launch" to start browsing
